## 🎯 C'est quoi ?

**Petit Nom** est une app web pour les futurs parents qui cherchent le prénom idéal. Inspirée de Tinder, elle permet de **swiper** sur des prénoms, chacun de son côté, et de découvrir vos **matchs** — les prénoms que vous aimez tous les deux. 💕

## ✨ Fonctionnalités

- 🔍 **Découverte par swipe** — Parcourez les prénoms en swipant à droite (❤️) ou à gauche (✕)
- 👨👩 **Mode Papa & Maman** — Chacun fait ses choix indépendamment
- 💕 **Matchs automatiques** — Quand Papa et Maman aiment le même prénom, c'est le feu d'artifice !
- 🧠 **Suggestions intelligentes** — L'app apprend vos goûts et propose des prénoms similaires
- ⭐ **Notation 5 étoiles** — Classez vos favoris par préférence
- 📊 **Statistiques** — Origines préférées, styles, sonorités…
- 🌍 **20 origines culturelles** — Français, Hébreu, Arabe, Grec, Latin, Celtique, Japonais, Coréen, Polynésien…
- 🏷️ **13 tags de style** — Classique, Moderne, Doux, Fort, Royal, Mythologique, Mixte…
- ⚙️ **Filtres avancés** — Origine, popularité, longueur, initiale, sonorité
- 🔤 **Vérification des initiales** — Alerte si les initiales forment un acronyme gênant
- 🌙 **Mode sombre**
- 📤 **Export WhatsApp** — Partagez votre liste en un clic

## 🛠️ Stack technique

- **Frontend** : HTML/CSS/JS vanilla (single-page app, un seul fichier)
- **Backend** : [Supabase](https://supabase.com) (PostgreSQL gratuit)
- **Hébergement** : [GitHub Pages](https://pages.github.com)
- **Fonts** : Cormorant Garamond + Nunito (Google Fonts)
